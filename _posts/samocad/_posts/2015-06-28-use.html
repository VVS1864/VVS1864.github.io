Windows

Чтобы запустить программу на Windows, необходимо:
  • 1) Скачать zip архив с исходниками
  • 2) Распаковать архив
  • 3) Скачать и установить:

    Python 2.7 x86 или x64

    wxPython x86 или x64

    PyOpenGL x86 или x64

    Numpy x86/x64

    PIL x86/x64

    Reportlab x86 или x64

  • 4) Перейти в папку с программой и запусть файл samocad_start.py

    • Linux Ubuntu/Debian-like

    Выполнить в терминале:
    Установить необходимые пакеты:
    Debian 8 Jessie:
    sudo apt-get install python-wxgtk3.0 python-opengl python-numpy python-imaging python-reportlab
    Debian 7 Wheezy:
    sudo apt-get install python-wxgtk2.8 python-opengl python-numpy python-imaging python-reportlab
    Скачать, распаковать и запустить программу:
    wget https://github.com/VVS1864/SAMoCAD/archive/wx_opengl.zip
    unzip wx_opengl.zip
    cd SAMoCAD-wx_opengl
    python samocad_start.py
    Have fun!
    На компьютере должен быть OpenGL не менее версии 1.4.
