Download SAMoCAD:

Github - Most new version (updated frequently)

Sourceforge - An older version may be out of date (rarely updated)

SAMoCAD written in Python 2 and requires no installation. However, you need to have installed Python 2.7 Download Python.

Unpack the downloaded archive, and run the file SAMoCAD.pyw.
For Linux:

unzip SAMoCAD-master.zip
cd SAMoCAD-master
python2.7 SAMoCAD.pyw

You can work:)
