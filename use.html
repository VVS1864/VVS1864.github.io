Скачать SAMoCAD:

Github - Наиболее свежая версия программы (часто обновляется)

Sourceforge - Более старая версия, может быть сильно устаревшей (обновляется редко)

SAMoCAD написан на Python 2 и не требует установки. Однако у вас должен быть установлен Python 2.7 Скачать Python.

Скачанный архив нужно распаковать (в Windows в пути к папке с программой не должно быть русских символов) и запустить файл SAMoCAD.pyw.
Для Linux:

unzip SAMoCAD-master.zip
cd SAMoCAD-master
python2.7 SAMoCAD.pyw

Можно работать:)
